# DRUPELET

The [kernel of a raspberry](https://en.wikipedia.org/wiki/Drupe).

> In botany, a drupe (or stone fruit) is a type of fruit in which an outer
> fleshy part (exocarp, or skin, and mesocarp, or flesh) surrounds a single
> shell (the pip (UK), pit (US), stone, or pyrena) of hardened endocarp with a
> seed (kernel) inside. 
>
> ...
>
> In an aggregate fruit, which is composed of small, individual drupes (such as
> a raspberry), each individual is termed a drupelet, and may together form an
> aggregate fruit.

Drupelet is a minimal Unix-like kernel targeting AArch64. It implements a
multi-level feedback queue scheduler, virtual memory (4KB pages) via
multi-level page tables, and an inode-based file system.
